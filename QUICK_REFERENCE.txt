================================================================================
  TROUBLESHOOTING
================================================================================
Issue: "Connection refused" on localhost:3001
  ‚Üí Make sure backend is running: npm run dev in server folder
  
Issue: "Alchemy API key not working"
  ‚Üí Check .env file in server folder has your actual key
  ‚Üí Restart backend after changing .env
  
Issue: Frontend shows errors
  ‚Üí Clear browser cache: Ctrl+Shift+Delete
  ‚Üí Restart frontend: npm start
  ‚Üí Check browser console: F12
  
Issue: Email alerts not working
  ‚Üí Make sure backend is running on port 3001
  ‚Üí Check browser console for network errors

================================================================================
  ENVIRONMENT SETUP
================================================================================
Backend .env (c:\Users\Darshan\Desktop\web3\server\.env):
  ALCHEMY_API_KEY=your_key_here
  ALCHEMY_ENDPOINT_SEPOLIA=https://eth-sepolia.g.alchemy.com/v2/
  ALCHEMY_ENDPOINT_MAINNET=https://eth-mainnet.g.alchemy.com/v2/
  PORT=3001
  NETWORK=sepolia

Frontend .env (c:\Users\Darshan\Desktop\web3\mev-detector\.env):
  REACT_APP_API_URL=http://localhost:3001/api
  REACT_APP_WS_URL=ws://localhost:3001
